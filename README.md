#Simple ATM Management System 

##Overview

The ATM Management System is a simple console-based application written in the C programming language.
It simulates basic ATM operations such as checking account balance, depositing money, and withdrawing money.

This project is designed for beginners who want to understand how real-world systems like ATMs work using fundamental C programming concepts.

##Objectives
 
*To understand the use of functions in C.
*To practice conditional statements.
*To implement a menu-driven program.
*To simulate basic banking operations.
*To improve problem-solving and logical thinking skills.

##Features
 
*Check current account balance.
*Deposit money into the account.
*Withdraw money with balance validation.
*Displays message for insufficient balance.
*Simple and user-friendly menu system.

##Concepts Used

This project demonstrates the following C programming concepts:

*Global variables.
*Functions.
*Conditional statements (if-else).
*User input using scanf().
*Output using printf().
